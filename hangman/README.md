# Hangman Game

The classic word-guessing game with beautiful ASCII art, 6 lives, and dramatic win/lose screens.

### Features
- Random word from a built-in list of 30+ fun words
- Full 7-stage hanging animation (ASCII art)
- Prevents repeated guesses
- Clear lives counter and visual feedback
- Dramatic win/lose messages
- 100% console-based, zero dependencies
