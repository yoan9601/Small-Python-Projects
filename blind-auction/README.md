# Blind Auction

Secret bidding program – the highest bidder wins!  
Perfect demonstration of dictionaries, loops and clean UX.
$120 → $150 → $301 → Winner: Alex with $301!

### Features
- Unlimited participants
- Secret bids (others can't see)
- Automatic winner detection
- Screen clearing between bids
- Beautiful ASCII logo
- Works on Windows, Mac & Linux

### Live Demo
